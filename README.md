# APA7ish Quarto templates

## Installing

Use quarto-apaish in an existing Quarto project

```bash
quarto add mvuorre/quarto-apaish
```

Create a new Quarto project using quarto-apaish:

```bash
quarto use template mvuorre/quarto-apaish
```
